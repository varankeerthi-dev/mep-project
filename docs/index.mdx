# MEP ERP

Cloud-based quoting platform for Mechanical, Electrical & Plumbing contractors.

## Modules

- **Quotations** — Create, manage, and revise BOQ-based quotations
- **Delivery Challans** — Track material dispatch and link to quotations
- **Invoices** — Generate invoices from quotations and DCs
- **Projects** — Manage project scope, site visits, and milestones
- **Materials** — Centralised item master with variants, pricing, and stock
- **Clients** — Client management with discount portfolios
- **HR** — Employee directory, attendance, and payroll

## Tech Stack

- **Frontend:** React + Vite + TypeScript
- **Backend:** Supabase (PostgreSQL + REST API)
- **Auth:** Supabase Auth (email + Google SSO)
- **Package Manager:** pnpm workspaces (monorepo)
